SONR Music 2 flasher - macOS
============================

NOTHING TO INSTALL. No driver, no certificate, no admin.
WebUSB works out of the box in Chrome or Edge on macOS.

How to flash:
  1. Open  https://ygrikbetatest.github.io/sonr-music2-flasher/  in Chrome or Edge.
  2. Hold "+", plug in the USB cable, make sure the LED is OFF (board in BOOT).
  3. Press Update. If a device-picker appears, select "SONR ... Bootloader" and click Connect.
  4. When prompted, press "+" once. Wait for the confetti.

You may see the browser's device-picker once or twice - that is normal on macOS and does
not stop the flash. To make it fully picker-free (optional), a managed browser policy can be
installed via a configuration profile - ask if you want it; it is not required to flash.
